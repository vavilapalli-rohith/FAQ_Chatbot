# 🤖 AI FAQ Chatbot

An AI-powered FAQ Chatbot developed using Python, Natural Language Processing (NLP), and Tkinter. The chatbot intelligently understands user queries and provides the most relevant responses using TF-IDF Vectorization and Cosine Similarity techniques. It features an interactive graphical user interface with a modern dark theme, chat history, and real-time responses.

---

## 📌 Features

* AI-powered FAQ response system
* Natural Language Processing (NLP)
* Intelligent question matching using Cosine Similarity
* Interactive and user-friendly GUI
* Dark Theme Interface
* Real-time chatbot responses
* Clear Chat functionality
* Supports greetings and general conversations
* Handles technical, educational, and company-related FAQs
* Press Enter key to send messages



## 🧠 How It Works

1. The user enters a question.
2. The chatbot preprocesses the input text.
3. TF-IDF converts questions into numerical vectors.
4. Cosine Similarity calculates the similarity score between the user's question and the stored FAQs.
5. The chatbot finds the best matching question.
6. The most relevant response is displayed instantly.
7. If no suitable match is found, the chatbot returns a default response.
